_________________________     PDF-Downloader     ____________________________



				SUMMARY:

PDF-Downloader is used to download PDF-files from website links in an Excel
file. It automates the download process, and it handles errors like if a 
link doesn't contain a PDF-file.


			INSTALLATION (WINDOWS):

To run the program on Windows machines, you need a program that can run 
Python (.py) files, e.g. Visual Studio Code.


			INSTALLATION (LINUX):

Most of the time a Linux distribution comes with Python pre-installed. 
If not, then you will have to install it. You run the program from the 
terminal, by writing: "python pdf-downloader.py" and press return.


			>>>>>>IMPORTANT<<<<<<

BEFORE USING THE PROGRAM YOU NEED TO READ THE CODE AND CHANGE THE VALUES
AS DESCRIBED IN THE COMMENT SECTIONS OF THE CODE! THIS IS IMPORTANT FOR THE
PROGRAM TO WORK CORRECTLY.
ALSO, IF THE PROGRAM WON'T RUN, YOU NEED TO INSTALL THE PACKETS LISTED AT THE
TOP OF THE CODE (SEE THE "IMPORT..." SECTION).
WARNING: IF YOU SET THE "i" VALUE TOO HIGH (PRE SET TO "5") IT MIGHT CAUSE
A NETWORK CONGESTION! SO BE CAUTIOUS.
